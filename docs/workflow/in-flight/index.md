# In-Flight

Active work tracked in the session banner. Each item has an `index.md` with frontmatter (`status: active`) and a `tasks.md` for agent execution checklists.

- **Initiatives** — strategic goals (months), contain nested epics
- **Epics** — tactical work (weeks), contain nested items
- **Items** — discrete deliverables (days)

Completed items move to `archive/` subfolders.
